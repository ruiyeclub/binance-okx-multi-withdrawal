## 使用说明：

1、git clone https://github.com/ruiyeclub/binance-okx-multi-withdrawal.git

2、cd binance-okx-multi-withdrawal

3、pip install -r .\requirements.txt

4、配置说明： 
> a、.env说明: 这里放置交易所提币相关配置
 
> b、wallets.txt说明: 这里放置需要批量提币的钱包地址
 
> c、运行main.py
